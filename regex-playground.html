Position Anchors

^ Start of string
$ End of string
\b Word boundary
\B Not word boundary

Character Classes

[abc] Any of a, b, c
[^abc] Not a, b, or c
[a-z] a through z
[0-9] Digits 0-9
\d Any digit
\D Not a digit
\w Word character
\W Not word character
\s Whitespace
\S Not whitespace

Quantifiers

* 0 or more
+ 1 or more
? 0 or 1
{3} Exactly 3
{3,} 3 or more
{3,5} 3 to 5

Groups

(abc) Capture group
(a|b) a or b
(?:abc) Non-capturing
(?=abc) Lookahead
(?!abc) Negative lookahead

Special Characters

. Any character
\. Literal dot
\\ Literal backslash
\n Newline
\t Tab
